# Kamio AI System

このプロジェクトは、神尾信也の意思決定・思考・事業設計を再現し、事業開発を高速化するためのAI環境です。

## 目的
- 事業開発の高速化
- 意思決定の再現
- Codexによる自動実行

## ルール
- 実装前提で考える
- 抽象ではなく具体
- スピード重視

## 最初に読むファイル
- /context/kamio_profile.md
- /context/decision_rules.md
- /context/writing_style.md
